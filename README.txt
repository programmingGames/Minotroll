## DISTRIBUIÇÃO DO PROGRAMA

    - O programa se encontra inteiramente distribuido por modulos, em que cada 
    modulo contem uma ou mais classes que representam um ou mais elementos dentro
    do programa.

    ## DISTRIBUIÇÃO DE PASTAS
    
        As pastas so jogo encontram-se distribuidos da seguinte forma:
            - .vscode: informações sobre o editor utilizado referentes ao projeto
            - data: pasta que contem todos os modulos do jogo;
            - resources: pasta que contem todos os recursos visuais e sonoros do jogo;
            - user: pasta que contem pastas referentes aos usuarios cadastrados no jogo;
            - main.py: ficheiro python pelo qual é possivel rodar o jogo.

*************************************************************************************

## COMO RODAR O PROGRAMA/JOGO

    O jogo pode ser rodado com o interpretador python em um computador, desde que 
    este tenha a biblioteca pygame instalada no mesmo. E o jogo deve ser rodado 
    atravez do ficheiro main.py ou também uma outra forma de rodar o jogo seria 
    executando o ficheiro Minotrol.exe caso o sistema operativo seja Windows, sendo
    que o Windows Defender tem de estar desligado (porque se não este ira para a 
    execução do jogo, e apagar o ficheiro Minotrol.exe da pasta do projeto).

## AMBIENTE DE ELABORAÇÃO DO PROJETO

    O projeto foi quase inteiramente elaborado sobre o sistema operativo Linux, mas 
    acompanhado com testes de execução no sistema operativo Windows.

## INFORMAÇÕES SOBRE O EXECUTAVEL

    O ficheiro executavel do jogo foi feito para computadores Windows utilizando
    uma biblioteca do python chamada pyinstaller. Este pode executado e os componentes
    dp jogo seram monstrados em uma tela.
